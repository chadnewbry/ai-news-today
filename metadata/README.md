# App Store Metadata

This directory contains App Store Connect metadata for AI News Today.

## Files

- **app_store.json** — Title, subtitle, keywords, description, category, age rating, and URLs

## Keyword Strategy

Keywords are optimized for discoverability in the News and AI/tech space:

| Keyword | Rationale |
|---------|-----------|
| ai news | Primary search term, highest intent |
| daily digest | Core feature differentiator |
| news summary | Describes main value proposition |
| artificial intelligence | Long-tail, high-volume category term |
| tech news | Broad category capture |
| ai headlines | Intent-specific variation |
| news aggregator | Category descriptor |
| smart news | Competitive positioning term |
| personalized news | Key feature keyword |
| ai updates | Trending search term |
| machine learning news | Adjacent category capture |
| news briefing | Feature-aligned keyword |
| ai curated | Unique differentiator |
| news reader | Broad category term |
| trending ai | Timely, high-search-volume term |

**Total keyword length:** 98 characters (max 100)

## Notes

- Category: **News** (primary) — best fit for discovery in App Store Browse
- Age rating: **12+** — news content may include coverage of sensitive real-world events
- Description follows App Store best practices: feature bullets, clear value prop, CTA at end
